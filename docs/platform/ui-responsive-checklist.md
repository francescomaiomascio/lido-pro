# UI Responsive Checklist

Use this checklist for every wave that touches UI, layout, navigation, map, panels, forms, tables, modals, drawers, sheets, or typography.

## Must Pass

- No horizontal overflow.
- No clipped side content.
- No overlapped typography.
- Floating controls do not cover critical content.
- Bottom sheets and side panels scroll internally.
- Maps/canvases remain usable.
- Primary actions remain reachable.
- Touch targets remain usable.
- Tablet portrait is not a squeezed desktop layout.
- Phone portrait is verticalized.
- Phone landscape keeps navigation and primary actions reachable.
- Browser preview is not treated as final native validation.
