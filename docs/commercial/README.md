# Commercial Readiness Index

LidoPro is a proprietary commercial product in active development.

This index tracks commercial-readiness boundaries. It is documentation only and does not implement commercial backend behavior.

## Current Positioning

- Product: LidoPro.
- Repository: source-available/public-ready, not open source.
- Product state: pre-release / private commercial readiness.
- Runtime posture: local-first application.
- Commercial use: requires written permission or separate license.

## Required Before Wider Commercial Distribution

- customer terms;
- privacy policy;
- data processing/security posture;
- payment-provider setup and terms if Lido Pay is released;
- cloud/SaaS terms if Lido Cloud is released;
- support and maintenance model;
- commercial deployment checklist;
- release artifact/signing process;
- customer data handling policy;
- incident/security contact process.
