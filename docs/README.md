# LidoPro Documentation

This directory contains product, platform, repository, legal, commercial, architecture, and delivery-wave documentation for LidoPro.

## Index

| Directory | Purpose |
| --- | --- |
| `docs/brand/` | Product naming, module naming, visual identity boundaries, and brand/trademark posture. |
| `docs/product/` | Product scope, implemented capabilities, commercial roadmap boundaries, and protected layout rules. |
| `docs/platform/` | Runtime/platform notes for Tauri Desktop, Capacitor Android, browser preview, storage, macOS, and Linux. |
| `docs/repo/` | Repository structure, ownership map, hygiene rules, public/source-available policy, and release checklist. |
| `docs/legal/` | Commercial legal boundary, distribution states, and release authorization boundaries. |
| `docs/commercial/` | Commercial readiness index and later customer/product release requirements. |
| `docs/architecture/` | Local-first architecture and system design notes. |
| `docs/waves/` | Delivery wave reports and historical implementation notes. |
