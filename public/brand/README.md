# LidoPro Brand Assets

Use `svg/` for scalable brand placement and `png/` for exported bitmap use.

## SVG

| Asset | Use |
| --- | --- |
| `svg/lidopro-wordmark-color.svg` | Primary color wordmark |
| `svg/lidopro-wordmark-black.svg` | Single-color dark wordmark |
| `svg/lidopro-wordmark-white.svg` | Single-color light wordmark |
| `svg/lidopro-mark-color.svg` | Primary color standalone mark |
| `svg/lidopro-mark-black.svg` | Single-color dark standalone mark |
| `svg/lidopro-mark-white.svg` | Single-color light standalone mark |

## PNG

Each SVG has transparent PNG exports in `png/`:

- `*.png`: 1x export
- `*@2x.png`: 2x export

## Runtime Aliases

Root-level files remain for existing app/runtime references:

- `lidopro-wordmark-transparent.png`
- `lidopro-mark-transparent.png`
- `lidopro-wordmark-traced.svg`
- app icons and favicon files

Prefer the organized `svg/` and `png/` folders for new brand usage.
