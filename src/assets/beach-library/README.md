# LidoPro Asset Library

Questa cartella contiene asset grafici importati per la libreria Canvas.

Regole:
- ogni asset deve avere licenza verificata;
- ogni asset deve essere registrato in manifest.json;
- non aggiungere file senza fonte;
- preferire CC0/Public Domain/MIT/Apache-2.0;
- non usare asset con licenza incerta;
- non usare immagini raster pesanti;
- non modificare layout o dati operativi.

Stato MAP.R6.3:
- gli asset sono preview/libreria;
- non sostituiscono ancora il renderer operativo della mappa;
- non salvano scelte nel database.
