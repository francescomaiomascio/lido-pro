# MAP.R1 - Canvas Foundation

## Scopo

Creare le fondamenta del futuro motore Canvas parametrico per la mappa spiaggia.

## Decisione

Canvas 2D e il motore grafico principale.
Svelte/HTML resta per pannelli, toolbar, menu e controlli.

## Incluso in MAP.R1

- pulizia menu Spiaggia
- rimozione Filtri dal menu
- nuove sezioni future sotto Spiaggia
- tipi base Canvas
- camera model
- conversioni metriche
- griglia metrica
- snapping
- default LidoPro 31m x 28m

## Escluso

- sostituzione mappa attuale
- modifica seed
- modifica beach_items
- migrazione DB
- editor completo
- drag
- salvataggio layout
